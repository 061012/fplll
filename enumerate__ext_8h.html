enumf *  rdiag,
enumf *  pruning

Callback function given to external enumeration library.

You have to pass a pointer to an array 'enumf mu[mudim][mudim]'. When flag mutranspose is true then mutransposed is actually stored there. You have to pass a pointer to an array 'enumf rdiag[mudim]' and an array 'enumf pruning[mudim]'. Note: for dual SVP you also get mu and rdiag as is, so the external library must make the respective changes to Mu and Rdiag itself.
